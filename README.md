# RichHouse Game Bot

Игровая воронка продаж для RichHouse: ежедневная коробка, баллы, билеты, розыгрыши, заявки в Telegram и админка.

## MVP

- Next.js + TypeScript + Tailwind
- Supabase PostgreSQL
- Telegram Bot API webhook
- Админка для заявок, участников и розыгрышей

## Статус

Стартовая архитектура проекта.
